## Simple To-do app

### Tech

- Node.js (Express.js)
- MongoDB
- React

### Start Server

Create a mongodb database and add your connection string into `.env` file. (See sample file)

```
cd server
npm install
npm start
```

Server running on port 3001.

### Start Client

```
cd client
npm install
npm start
```

Open http://localhost:3000 to view it in the browser.
